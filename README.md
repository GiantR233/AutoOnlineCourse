'''自用挂课脚本'''<br>
项目概述<br>
网课自动化学习助手是一个基于Python和Selenium的智能自动化工具，专门设计用于自动化完成在线课程的学习任务。该项目能够模拟真实用户行为，自动处理视频播放、验证码识别、章节切换等复杂交互场景。<br>
<br>
项目背景<br>
在线教育平台通常要求学员完成固定的视频学习时长，但手动操作既耗时又重复。本项目通过自动化技术解决这一问题，让学习过程更加高效。<br>
<br>
核心功能<br>
1. 智能反检测系统<br>
随机化User-Agent，模拟真实浏览器指纹<br>
清除WebDriver自动化痕迹<br>
人类行为模拟（随机延迟、自然操作间隔）<br>
2. 自动化登录模块<br>
支持用户名密码登录<br>
自动识别并处理登录验证码<br>
使用ddddocr库进行图像验证码识别<br>
3. 课程播放管理<br>
自动点击播放按钮<br>
智能检测视频暂停状态并恢复播放<br>
根据视频时长自动等待完成<br>
4. 验证码处理系统<br>
实时检测验证码弹窗<br>
自动截图并识别验证码内容<br>
智能填写验证码并提交<br>
5. 章节导航<br>
自动跳转到下一章节<br>
处理课程进度跟踪<br>
<br>
技术栈<br>
技术类别	        具体技术	                                         用途<br>
核心框架	        Selenium WebDriver	                             浏览器自动化控制<br>
验证码识别	      ddddocr、pytesseract、PIL	                       图像验证码识别<br>
反检测技术	      随机User-Agent、属性重定义	                       绕过网站自动化检测<br>
网页交互	        显式等待、XPath定位、ActionChains	               稳定可靠的元素操作<br>
工具库	          re、random、time	                               数据处理和流程控制<br>
