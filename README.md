项目概述
网课自动化学习助手是一个基于Python和Selenium的智能自动化工具，专门设计用于自动化完成在线课程的学习任务。该项目能够模拟真实用户行为，自动处理视频播放、验证码识别、章节切换等复杂交互场景。

项目背景
在线教育平台通常要求学员完成固定的视频学习时长，但手动操作既耗时又重复。本项目通过自动化技术解决这一问题，让学习过程更加高效。

核心功能
1. 智能反检测系统
随机化User-Agent，模拟真实浏览器指纹
清除WebDriver自动化痕迹
人类行为模拟（随机延迟、自然操作间隔）
2. 自动化登录模块
支持用户名密码登录
自动识别并处理登录验证码
使用ddddocr库进行图像验证码识别
3. 课程播放管理
自动点击播放按钮
智能检测视频暂停状态并恢复播放
根据视频时长自动等待完成
4. 验证码处理系统
实时检测验证码弹窗
自动截图并识别验证码内容
智能填写验证码并提交
5. 章节导航
自动跳转到下一章节
处理课程进度跟踪
